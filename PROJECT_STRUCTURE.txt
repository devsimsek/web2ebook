Quick Start:
1. make install           # Creates venv, installs deps
2. make test              # Verify installation
3. make run URL=<url>     # Convert a page

Features:
✓ Single page conversion
✓ Multi-page crawling (--crawl)
✓ EPUB, PDF, MOBI formats
✓ Auto cover generation
✓ Image embedding
✓ Clean page titles
✓ Code block styling
✓ Smart HTML filtering
